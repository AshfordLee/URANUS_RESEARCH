## 环境要求

- Python 3.6 或更高版本
- pandas >= 1.3.0
- numpy >= 1.20.0

## 安装步骤

1. 克隆或下载项目到本地：
```bash
git clone <项目地址>
cd <项目目录>
```

2. 创建并激活虚拟环境（推荐）：
```bash
python -m venv .venv
source .venv/bin/activate
```

3. 安装依赖：
```bash
pip install -r requirements.txt
```

## 运行方法

1. 确保已激活虚拟环境（如果使用）：
```bash
source .venv/bin/activate
```

2. 运行程序：
```bash
python question1.py
python question2.py
```

3. 根据提示输入矩阵大小n（正整数）
